CSS Anchor Positioning

position-try-fallbacks

ボタンをクリックすると、それぞれ異なる位置にポップオーバーが表示されます。

position-area: block-end

ボタンの下に表示されます

コード例

/* アンカーを定義 */ #anchor-top { anchor-name: --anchor-top; } /* ポップオーバーの位置を指定 */ .popover-top { position-anchor: --anchor-top; position-area: block-start; /* はみ出したら上下を反転 */ position-try-fallbacks: flip-block; }
